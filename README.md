Set up:
1. Open Assignment3 project folder on Intellij IDEA
2. Open up postgres
3. Create a new database named "Students"
4. Open up the query tool on the newly created database
5. Run the database creation scripts in the query tool found in the file "CreationScripts.txt" that is found within the Assignment3 folder
6. Right click on PostGreSQL (version number) above all of the databases and click properties
7. Go to connection and copy the port number and replace the port number in Main.java
8. Also replace the user and password with your user and password
9. Now run Main.java and the application should work
