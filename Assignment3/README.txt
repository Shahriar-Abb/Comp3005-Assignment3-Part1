Set up:
1. Open up postgres
2. Create a new database named "Students"
3. Open up the query tool on the newly created database
4. Run the database creation scrips in the query tool found in the file "CreationScripts.txt"
5. Right click on PostGreSQL (version number) above all of the databases and click properties
6. Go to connection and copy the port number and replace the port number in Main.java
7. Also replace the user and password with your user and password
8. Now run Main.java and the application should work
